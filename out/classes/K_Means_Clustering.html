K_Means_Clustering Class

Defined in: K_Means.js:10

The K_Means_Clustering class contains all the necessary logic for clustering unsupervised data.

Constructor

K_Means_Clustering

(
  • args
)

Defined in K_Means.js:10

Parameters:

  • args Object

    Contains all the necessary parameters for K-Means clustering as listed below.

    • random_Init_Count Number

      Required parameter for how many times to randomly initialize centroids.

    • cluster_count Number

      Required value to specify how many clusters in total are to be initialized.

    • notify_count Number

      Required value to execute the iteration_callback after every x number of iterations.

    • iteration_callback Function

      Required callback that can be used for getting information for diagnostics.
